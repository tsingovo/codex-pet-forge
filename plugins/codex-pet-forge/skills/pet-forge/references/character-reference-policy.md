# Character reference policy

- Use the uploaded image for identity, palette, clothing, hair, face, and style.
- Ignore scenery, watermarks, UI, captions, and unrelated people.
- If a second image is an expression reference, copy only the requested expression language, not its character identity.
- Normalize the subject into a compact full-body pet readable inside `192x208`.
- When the source is face-only or bust-only, preserve visible identity cues and infer unseen clothing and footwear conservatively; avoid complex invented props.
- Prefer simple opaque shapes and crisp edges.
- Do not add logos or readable text.
- Do not redistribute user-provided or generated character art in this plugin repository.
